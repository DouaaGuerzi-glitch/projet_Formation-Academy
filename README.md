# Personne 4 – JavaScript, Base de données et Suppression

Cette partie regroupe la gestion dynamique, la base de données
et les opérations de suppression.

## Fichiers réalisés
- main.js : interactions et validation côté client
- delete.php : suppression des données
- database/ : fichiers SQL de la base de données

## Technologies utilisées
- JavaScript
- PHP
- MySQL

## Fonctionnalités
- Validation des formulaires côté client
- Suppression des enregistrements
- Création et gestion de la base de données

## Rôle
Responsable du dynamisme, de la base de données et des opérations critiques.
